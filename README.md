# Planet one

Planet one site

## Installation

Use the [node js (LTC)](https://nodejs.org/en/) to install.

```bash
npm i
```

## Build for dev

```bash
gulp dev
```

## Build for production

```bash
gulp build
```

## Run app

```bash
gulp watch serve
```

Runs the app in the development mode.
Open http://localhost:3000 to view it in the browser.
