Как добавлять новые отзывы на лендинг. 

1) залогиниться на хостинге https://sprutio.beget.com/ 
2) перейти в следующие папки:
andretps.beget.tech -> public_html -> static -> js
3) открыть файл под названием "reviews"
4) скопировать последний отзыв, заполнить по образцу: 
- name - имя
- date - дата отзыва (если нет, оставить пустые кавычки)
- photo - ссылка на фото (если нет, оставить пустые кавычки)
- content - текст отзыва. 

текст должен идти без разрывов строк. при необходимости переноса текста - поставить знак <br>. 
пример: 
{
    "content": "Мороз и солнце.<br>День чудесный!"
}

Текст не должен содержать двойных кавычек. При необходимости использовать их, необходимо перед ними поставить знаки 
экранирования (см пример ниже)
пример:
{
    "content": "Здесь текст без кавычек. \"А вот здесь уже в кавычках!\" А здесь снова без кавычек!"
}
